Abstract

Datasets


Multimodal Microscopy Dataset (Gu et.al, IEEE TMI 2018)
This dataset contains the endomicroscopy and H&E histology data from 42 patients. We also recommend to use BreakHis dataset and PLOSONE dataset to improve the crossmodal generalization.
